# Maayan Shani Portfolio

A clean one-page portfolio website based on Maayan Shani's resume content.

## Included sections
- Hero
- About
- Skills
- Selected Projects with GitHub buttons
- Education
- Contact
- Resume PDF button

## Files
- `index.html` – website structure and content
- `style.css` – responsive styling
- `script.js` – small mobile navigation script
- `Resume.pdf` – resume file opened by the View Resume buttons

## Updating project links
In `index.html`, search for `View on GitHub` and replace each `href` value with the exact GitHub repository URL.

Example:
```html
<a class="project-link" href="https://github.com/Maayan-Shani/BrainShare" target="_blank" rel="noreferrer">View on GitHub</a>
```

## How to use
Open `index.html` in a browser.

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files to the repository.
3. Go to Settings → Pages.
4. Select the main branch and root folder.
5. Save and wait for the GitHub Pages link.
